# Frontier ideation branch (YC W26 signal + RFS + a16z Big Ideas 2026)

This directory is a **research synthesis branch**: public web pages were fetched (no authentication, no paywall bypass) and distilled into strategic clusters and a **value proposition portfolio**. It does **not** replace the frozen `phase2` blueprint corpus; it is an **upstream signal layer** for Idea Browser operators.

## Contents

| File | Description |
|------|-------------|
| `SOURCES_CRAWL_LOG.md` | URLs retrieved, limitations (JS directory pages), crawl ethics note |
| `YC_RFS_SUMMER2026_SYNTHESIS.md` | Themes extracted from YC Requests for Startups (Summer 2026 section) |
| `A16Z_BIG_IDEAS_2026_SYNTHESIS.md` | Themes from a16z Big Ideas 2026 Part 1 and Part 2 |
| `W26_BATCH_CLUSTER_MAP.md` | How W26 companies cluster; official vs third-party sourcing |
| `VALUE_PROPOSITION_PORTFOLIO.md` | **Primary deliverable**: portfolio of ideation value propositions |

## How to use

1. Treat `VALUE_PROPOSITION_PORTFOLIO.md` as a **prioritization menu** for new blueprints or research envelopes.  
2. Cross-check any third-party company list against **YC Bookface**, **official Launch YC posts**, or `ycombinator.com/companies` once batch filters render in your browser.  
3. When importing into `phase2`, assign fresh `blueprint_id` values and run schema validation.
